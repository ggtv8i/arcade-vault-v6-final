══════════════════════════════════════════════════════════════════
🌐 الاستضافة على هوست ألعاب (itch.io أو أي هوست)
══════════════════════════════════════════════════════════════════

  1. ضع ألعابك في المجلدات الصحيحة
  2. شغّل: python scan.py  (يولّد roms.json)
  3. ارفع كل الملفات (بما فيها systems/ كاملاً)
  4. افتح arcade-vault.html على الهوست
  5. اضغط "مسح وتحديث الألعاب"

══════════════════════════════════════════════════════════════════
💻 التشغيل المحلي (جهازك)
══════════════════════════════════════════════════════════════════

  Windows: انقر نقراً مزدوجاً على start.bat
  Mac/Linux: chmod +x start.sh && ./start.sh
  ثم افتح: http://localhost:8080/arcade-vault.html

  ⚠️ لا يعمل بفتح arcade-vault.html مباشرة (file://)
     يحتاج خادم HTTP (start.bat يوفره بضغطة واحدة)
